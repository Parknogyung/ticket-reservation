# 📅 Project Changelog (일자별 변경 내역)

## 2026-01-29
- **MSA 인프라 구축**: Spring Cloud 기반의 Eureka Server(서비스 디스커버리), Config Server(설정 중앙화), Gateway Server(API 게이트웨이) 추가 및 마이크로서비스 통합 작업 수행.

## 2026-01-26
- **대기열 시스템(Queue) 구현**: 대규모 트래픽 제어를 위한 대기열 진입 및 관리 로직 추가.

- **로그 모니터링 구축**: Logback과 Elasticsearch를 연동하여 실시간 로그 수집 및 모니터링 환경 개발.

## 2026-01-18
- **티켓 다중 예매 기능**: 한 번에 여러 장의 티켓을 예매할 수 있도록 기능 확장.
- **카카오 로그인 연동 강화**: 카카오 계정으로 로그인 시 티켓 예매 인증이 가능하도록 로직 수정.
